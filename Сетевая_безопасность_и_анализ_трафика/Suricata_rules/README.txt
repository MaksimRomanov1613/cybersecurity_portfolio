# Правила Suricata для детектирования вредоносного трафика

Четыре сигнатуры, написанные для обнаружения конкретных вредоносных пакетов в учебном дампе `malicious_traffic.pcap`.

## Правила
- **EvilBot User-Agent** (sid:1000001): детектирует HTTP-запросы с User-Agent "EvilBot"
- **DNS запрос на зловредный домен** (sid:1000002): отслеживает DNS-запросы к `malicious-domain.com`
- **C2 Handshake** (sid:1000003): фиксирует TCP-соединения на порт 4444 с текстом "C2: handshake"
- **ICMP-туннель** (sid:1000004): обнаруживает ICMP-пакеты, содержащие `ICMP_TUNNEL_123456`
