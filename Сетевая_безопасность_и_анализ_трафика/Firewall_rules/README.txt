На одном из pfSense создал несколько Firewall rules для конкретных задач безопасности. 
Заблокировал доступ к RDP из локальной сети, ограничил работу SMTP только рабочим временем (с 7:00 до 22:59) для одного выбранного дня, и полностью заблокировал возможность SSH-подключений из локальной сети. 
Экспортировал итоговую конфигурацию в файл fw_rules.xml. 
Научился формализовывать политики безопасности в правила NGFW, использовать расписания для временных ограничений и управлять доступом на уровне портов и протоколов.
