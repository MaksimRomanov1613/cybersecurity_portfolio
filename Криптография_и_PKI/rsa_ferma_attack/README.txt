# Атака Ферма на RSA

Реализация метода факторизации Ферма для модуля RSA, когда простые множители p и q близки друг к другу.  
Скрипт находит p и q по известному N, вычисляет закрытую экспоненту d и расшифровывает зашифрованное сообщение.

## Почему это работает
Если p и q близки, то N = p*q можно факторизовать перебором целых чисел от √N вверх.  
Метод ищет такие A и B, что A² - N = B², тогда p = A - B, q = A + B.


## Что делает скрипт
1. Принимает зашитые в код значения `n`, `e`, `c` (модуль, открытая экспонента, шифртекст).
2. Факторизует `n` методом Ферма.
3. Вычисляет функцию Эйлера φ(n), закрытую экспоненту d = e⁻¹ mod φ(n).
4. Расшифровывает `c`: m = c^d mod n.
5. Преобразует число в байты и выводит исходный флаг.


## Примечание
В учебном задании N специально сгенерирован с близкими p и q, чтобы атака работала.  
В реальных системах такое недопустимо – ключи RSA должны иметь большую разницу между множителями.
