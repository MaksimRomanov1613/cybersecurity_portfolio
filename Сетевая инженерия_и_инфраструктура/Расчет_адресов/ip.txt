2.1 ТЕХНОЛОГИЧЕСКИЙ СЕГМЕНТ (3 машины)
-----------------------------------------------------------------
Требуется: 3 IP-адреса для устройств.
В каждой подсети обязательно резервируются:
- один адрес для идентификатора сети (Network ID);
- один адрес для широковещательной рассылки (Broadcast).
Итого минимальное количество адресов = 3 + 2 = 5.

Ближайшая степень двойки, вмещающая 5 адресов, - 2^3 = 8.
Значит, общий размер подсети - 8 адресов.
Из них для хостов доступно: 8 - 2 = 6 адресов.

Формула расчёта маски:
Количество бит на хостовую часть = log2(8) = 3.
Префикс маски = 32 - 3 = 29.

Маска подсети: /29 (или 255.255.255.248).

Обоснование: Выбор маски /29 обеспечивает ровно 6 доступных адресов для устройств, что достаточно для трёх машин с запасом на возможное расширение до 6 устройств без изменения адресного плана.

-----------------------------------------------------------------
2.2 СЕРВЕРНЫЙ СЕГМЕНТ (16 машин)
-----------------------------------------------------------------
Требуется: 16 адресов для устройств.
С учётом адреса сети и broadcast: 16 + 2 = 18 адресов.

Ближайшая степень двойки: 2^5 = 32 адреса.
Хостов доступно: 32 - 2 = 30.

Количество бит на хосты = 5.
Префикс = 32 - 5 = 27.

Маска подсети: /27 (или 255.255.255.224).

Обоснование: Маска /27 позволяет адресовать до 30 хостов. Это минимально возможная маска, вмещающая 16 устройств, и оставляет запас на увеличение количества серверов до 30 без пересмотра IP-плана.

-----------------------------------------------------------------
2.3 ПОЛЬЗОВАТЕЛЬСКИЙ СЕГМЕНТ (32 машины)
-----------------------------------------------------------------
Требуется: 32 адреса для устройств.
С учётом адреса сети и broadcast: 32 + 2 = 34 адреса.

Ближайшая степень двойки: 2^6 = 64 адреса.
Хостов доступно: 64 - 2 = 62.

Количество бит на хосты = 6.
Префикс = 32 - 6 = 26.

Маска подсети: /26 (или 255.255.255.192).

Обоснование: Маска /26 предоставляет 62 адреса для хостов. Это минимальный вариант, вмещающий заданные 32 машины, с учётом служебных адресов и запасом на будущий рост (до 62 устройств).
