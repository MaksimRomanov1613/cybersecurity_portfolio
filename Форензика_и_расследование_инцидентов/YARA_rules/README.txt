# YARA-правила для идентификации вредоносных файлов

Три правила YARA, каждое из которых однозначно детектирует свой файл (file1.bin, file2.bin, file3.bin) без ложных срабатываний.

## Условия (согласно заданию)
- Каждое правило содержит минимум два условия: уникальная hex-строка и текстовая строка
- Правило для file2.bin дополнительно проверяет размер файла (`filesize < 600KB`)
