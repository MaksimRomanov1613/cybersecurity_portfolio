Разработал bash-скрипт, реализующий модель разграничения доступа по функциональным ролям. 
Создал пользователя user в группе default_users, а также трёх пользователей (secret_agent, secret_spy, secret_boss) в группе secret_users. 
Настроил права на домашние директории так, что члены secret_users могут заходить в каталоги друг друга, а остальные — нет. 
Сделал /var доступным всем на чтение и запись. Добавил установку apache2 с проверкой статуса сервиса. 
Для группы default_users настроил беспарольный доступ к sudo. 
Это дало понимание управления пользователями и группами в Linux, механизмов DAC (дискреционного управления доступом), настройки sudoers и автоматизации рутинных операций конфигурации сервера.
