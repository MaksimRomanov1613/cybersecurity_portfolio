# Инвентаризация и аудит Linux-системы

Bash-скрипт для сбора основной информации о системе. Запускается от root, собирает:
- список установленных пакетов (dpkg -l)
- запущенные процессы
- открытые порты (ss -tuln)
- версию ядра и операционной системы

Результат записывается в архив `OS_RESULT.tar`.
