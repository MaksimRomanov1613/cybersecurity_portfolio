Написал Python-скрипт, который с помощью библиотеки scapy формирует TCP-пакет с заданным текстом 
(«Dear Steel Cat! This is no attack, it's my humster Pinkie you should track») и отправляет его на локальный порт 127.0.0.1:12345. 
Затем с помощью Wireshark перехватил отправленное сообщение и выделил его в отдельный дамп sent_message.pcapng, содержащий только один этот пакет. 
Освоил ручную сборку TCP-сегментов, работу с сокетами на уровне scapy и методику верификации сгенерированного трафика через захват.
